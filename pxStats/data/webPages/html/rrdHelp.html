Required fields

Field Definition.
File Type Specify whether the sourlients are tx or rx.
Machines Specify from wich machine(s) the data concerning the sourlients is coming from.
Stats types Specify for wich data types graphics needs to be drawn.
Pre-determined spans Instead of specifying an hourly span, use this option to use pre-calculated spans.
Fixed spans When using pre-calculated spans, combine it with one of the fixed spans to get some real date intervals. Example : monthly + fixedprevious will draw up graph(s) that start on the first day of the previous month and end on the last day of the previous month.
Sourlients Specify the sourlient(s) for wich the the graphic(s) need to be drawn. To do so, use the window that will popup by clicking on the "add sourlients" button. The list of sourlients offered is dependant on the previously selected file type(s) and machine(s).

Advanced options (Optional fields)

End Time Select the date at wich the graphic is requested and ends. You can either fill out the text box manually or click on the calendar to select a date. Will use the current time as default value.
Group name Specify a group name to give to a series of sourlients that you want to combine.
Products List a series of specific products. Data present within the graphic(s) will be filtered as to contains information that is relative only to the specified product(s). Note : Products can contain wildcards.
Span Specify the time span of the graphic in hours. If no value is specified the default 24 hours value will be used.
Individual Specify that a graphics needs to be plotter for every data type and for every machines specified. Otherwise data will be combined between machines.
Total By using this option, the data of all the specified rx tx or sourlients will be combined to make a single entity.
