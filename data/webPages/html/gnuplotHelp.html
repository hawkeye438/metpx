Field Obligatory/Optional Definition.
End Time Optional. Select the date at wich the graphic is requested and ends. You can either fill out the text box manually or click on the calendar to select a date. Will use the current time as default value.
Group name Optional Specify a group name to give to a series of sourlients that you want to combine. Note : needs to be used with the combine option.
Products Optional List a series of specific products. Data present within the graphic(s) will be filtered as to contains information that is relative only to the specified product(s). Note : Products can contain wildcards.
Span Optional Specify the time span of the graphic in hours. If no value is specified the default 24 hours value will be used.
File Type Obligatory Specify whether the sourlients are tx or rx.
Machines Obligatory Specify from wich machine(s) the data concerning the sourlients is coming from.
Stats types Obligatory Specify for wich data types graphics needs to be drawn.
Sourlients Obligatory Specify the sourlient(s) for wich the the graphic(s) need to be drawn. To do so, use the window that will popup by clicking on the "add sourlients" button. The list of sourlients offered is dependant on the previously selected file type(s) and machine(s).
Combine the sourlients Optional Specify whether or not the graphics are to be drawn individually for each of the specified sourlients of if their data needs to be combined. Note : Becomes obligatory when using the group name option.
